FORMAS

PRISMAS

Áreas

Área da base (AB) = comprimento X altura da base Área lateral (AL) = perímetro da base X altura do prisma Área lateral (AT) = 2 X área da base X área lateral

Volume

Volume = Área da base X Altura

PIRÂMIDES

Áreas

Área da base (AB) = lado da base² Área lateral (AL) = perímetro da base X apótema Área lateral (AT) = ² X área da base X área lateral

Volume

Volume = 3/1 X área da base X altura

CILINDROS

Áreas

Área da base (AB) = π X raio² Área lateral (AL) = ² X π X raio X altura Área Total (AT) = 2 X área da base X área lateral

Volume

Volume = π X raio² X altura

CONES

Áreas

Área da base (AB) = π X raio² Área lateral (AL) = π X raio X geratriz Área Total (AT) = 2 X área da base X área lateral

Volume

Volume = 3/1 X π X raio² X altura

ESFERAS

Áreas

Área da superfíce (AS)= 4 X π X raio²

Volume

Volume = 3/4 X π X raio²

CONSIDERAÇÕES

  • Foram incluídas apenas uma das possíveis bases de pirâmides e prismas. A fórmula é diferente para cada base;
  • Apótema é a altura de um triângulo, medida de sua base até a vértice isolada;
  • Geratriz é a distância da vértice do cone até um ponto na circunferência da base, encontrada com:
    g² = h² + r²
  • h = altura;
  • b = base;
  • g = geratriz;
  • r = raio;
  • π = 3 ou 3,14 ou 3,14...;
