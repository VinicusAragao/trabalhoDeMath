formas

Prismas

Formulas das Áreas:

Área da base (AB) = comprimento X altura da base.

Área lateral (AL) = perímito da base X altura do prisma.

Área lateral (AT) = 2 X area da base X area lateral.

Volume:

Volume = Área da base X Altura

piramides

Formulas das Áreas:

Área da base (AB) = lado da base².

Área lateral (AL) = perímito da base X apótema

Área lateral (AT) = ² X area da base X area lateral.

Volume

Volume = 3/1 X área da base X altura

cilindros

Formulas das Áreas

Área da base (AB)= π X raio².

Área lateral (AL)= ² X π X raio X altura.

Área Total (AT)= 2 X Área da base X Área lateral.

Volume

Volume = π X raio² X altura

cones

Formulas das Áreas

Área da base (AB)= π X raio².

Área lateral (AL)= π X raio X geratriz.

Área Total (AT)= 2 X Área da base X Área lateral.

Volume

Volume = 3/1 X π X raio² X altura

esferas

Formulas das Áreas

Área da superfíce (AS)= 4 X π X raio².

Volume

Volume = 3/4 X π X raio²

Considerações

Depende da forma do prisma e da base da pirâmide para calcular a área da base.

Apótema é a altura de um triângulo da base até vértice da pirâmide;

Geratriz é a distância da vértice do cone até um ponto na circunferência da base, calculo:
g² = h² + r²

  • h = altura
  • b = base
  • g = geratriz
  • r = raio
  • π = pi ou 3 ou 3,14 ou 3,14..., depende da situação.
